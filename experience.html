My Experience

Background info:

I am a data scientist/analyst with experience in scientific programming, data visualization, and reporting. My goal is to work with data in an environment where the insights from my work directly impact the decisions being made with regard to business direction, product design, and content development.

Skills:

Python development

I have worked with Python since 2014, have taught graduate-level classes on Python, and written code for machine learning, analytics, data visualization, web development, and scientific research.

Statistical inference

I have used bayesian techniques and machine learning to make inferences and extract patterns from complex data. I have written code to analyze data for research, classwork, and as a freelance data analyst for a board game develpment company.

App development

I have developed web applications to manage databases, and display the data meaningfully and aesthetically.

Database management

I am comfortable with SQL and its various flavors. I am experienced in building database systems from provided data, data scraped from the web, and more.

Machine learning

I am comfortable with using popular machine learning techniqes to analyze a dataset. I can build Markov-Chain Monte-Carlo simulations to infer the patterns within data, convolutional neural networks to exploit the relationships between parameters, natural language processing algorithms to automate processes, and more.

Web crawling

I am not afraid to get my hands dirty and work hard to find data. I have used python packages such as Beautiful Soup and Selenium to crawl through hundreds of webpages for the data necessary to train my models.

Data visualization

With my background in academia, I am no stranger to creating beautiful plots to succinctly and aesthetically display data so that key inferences can be deciphered easily. I have used Matplotlib, Plotly, and other programs to create these figures.

Report writing

I enjoy writing, and am not afraid of it. I have prepared numerous professional reports in my time, including but not limited to scientific publications.
